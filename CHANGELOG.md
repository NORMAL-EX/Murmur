# 更新日志

## v0.1.0

首个发行版。

### 新增
- 消息发送图片(≤8MB,Markdown 内嵌渲染)
- 消息撤回:成员 40 秒内可撤回自己的消息,管理员 / 系统管理员可随时撤回任意人的消息;撤回内容不下发,普通用户与普通管理员仅见「此消息已被撤回」,仅系统管理员可「点击查看」原文
- 成员禁言(10 分钟 / 1 小时 / 1 天 / 7 天 / 永久,可随时解除)
- 管理员编辑用户资料(用户名、邮箱、头像、重置密码)
- 注册邮箱验证(6 位验证码 + 后台可配置 SMTP)
- 系统管理员私信审查台(查看任意成员私信往来、查看被撤回原文、可逐条撤回)
- WebSocket 弱网指数退避自动重连 + 心跳保活 + 断线补拉

### 优化
- 浅色模式搜索图标可见性修复;网络状态指示改为黑白
- 邮箱仅本人与管理员可见(其他用户不可见)
- 多架构镜像(linux/amd64 + linux/arm64)与基于 tag 的发行流程
